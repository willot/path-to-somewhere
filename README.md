# path-to-somewhere

A small dungeon crawler to demo React with nextjs and have some fun.
Use:
- React context
- tanstack query(ReactQuery)
- Code splitting
# to start 
Front-end
go to /path-to-somewhere and run npm run dev

to run the test use: npm run test

back end 
go to /server and run npm start
